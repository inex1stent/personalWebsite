# www.leoo.tech

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

## and open http://localhost:8080/.

see [configuration reference](https://cli.vuejs.org/config/).
